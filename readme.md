# Off-Road Terrain Semantic Segmentation using DINOv2

## Overview
Semantic segmentation model for off-road terrain classification using DINOv2 backbone and ConvNeXt-style segmentation head.

## Features
- Vision Transformer backbone (DINOv2)
- Fine-tuning strategy
- Cosine LR scheduler
- Mixed precision training
- Per-class IoU evaluation
- Visualization tools

## Results
- Initial IoU: 0.4996
- Improved IoU: ~0.50+

## Folder Structure
- src/ → Training & evaluation scripts
- models/ → Saved model weights
- results/ → Charts & outputs
- Dataset folders are ignored
